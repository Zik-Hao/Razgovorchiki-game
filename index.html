📜 Как играть

🎲 Режим «Тематические вопросы»

  • Настройки: Выберите интересующие темы (Воспоминания, Еда, Секс, Страхи и др.), количество пропусков и число игроков.
  • Игровой процесс: Нажимайте «Новый вопрос» для получения случайного вопроса из выбранных тем. Вопросы не повторяются, пока не закончатся.
  • Пропуск: Если вопрос не подходит, используйте кнопку пропуска (лимит задаётся в настройках).
  • По кругу: Включите этот режим, чтобы игроки отвечали по очереди, а не случайно.

🔥 Режим «Правда или Действие»

  • Шаг 1 — Выбор: Нажмите большую кнопку «🔥 ПРАВДА» или «⚡ ДЕЙСТВИЕ».
  • Шаг 2а — Правда: Вам выпадет честный вопрос из базы (~60 штук). Ответьте на него открыто.
  • Шаг 2б — Действие: Появится выбор из двух кнопок:
    • 🎲 Случайное задание — берётся из готовой базы (~60 заданий: весёлые, смелые, творческие).
    • 👥 Задание от игроков — остальные участники придумывают задание прямо сейчас, без использования списка.
  • Шаг 3 — Готово: После выполнения задания или ответа нажмите кнопку «✅ Готово», чтобы вернуться к выбору Правда/Действие для следующего раунда.
  • Игроки: Настройте количество участников и включите режим «По кругу» для поочерёдных ходов.

🛠 Общие функции

  • ❤️ Избранное: Сохраняйте понравившиеся вопросы и задания.
  • 📝 Заметки: Записывайте важные детали ответов. Экспортируйте в .txt файл через меню «Ещё».
  • 🏆 Достижения: Получайте бейджи за активность («Марафонец», «Философ», «Откровенный» и др.).
  • 🌙 Тёмная тема / 🔊 Звук: Настраиваются отдельно в каждом режиме.
  • 💾 Автосохранение: Прогресс сохраняется автоматически. Кнопка «Продолжить» на стартовом экране восстановит последнюю сессию.
